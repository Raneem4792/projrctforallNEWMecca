# مجلد مرفقات ردود الزائر السري

هذا المجلد مخصص لحفظ مرفقات ردود تقييمات الزائر السري.

## هيكل المجلد:
```
backend/uploads/mystery-responses/
├── README.md
└── (ملفات المرفقات ستُحفظ هنا تلقائياً)
```

## ملاحظات:
- الملفات تُحفظ بأسماء فريدة: `mresp_${timestamp}_${random}.${extension}`
- الحد الأقصى لحجم الملف: 10MB
- الحد الأقصى لعدد الملفات: 10 ملفات لكل رد
- الصيغ المدعومة: جميع صيغ الملفات

## إنشاء المجلد:
```bash
mkdir -p backend/uploads/mystery-responses
chmod 755 backend/uploads/mystery-responses
```

## الأمان:
- الملفات محمية من الوصول المباشر
- يتم التحقق من صحة الملفات قبل الحفظ
- يتم تنظيف أسماء الملفات من الأحرف الخطيرة
