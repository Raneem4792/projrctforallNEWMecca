# لوحة تحكم البلاغات - Dashboard

## نظرة عامة
هذا المجلد يحتوي على ملفات لوحة تحكم البلاغات منفصلة ومنظمة لتسهيل التعديل والصيانة.

## هيكل الملفات

```
dashboard/
├── dashboard.html      # ملف HTML الرئيسي
├── dashboard.css       # ملف CSS المخصص
├── dashboard.js        # ملف JavaScript
└── README.md          # هذا الملف
```

## الملفات

### 1. dashboard.html
- **الوصف**: الملف الرئيسي للصفحة
- **المحتوى**: هيكل HTML الأساسي مع تعليقات مفصلة
- **الميزات**:
  - شريط تنقل علوي مع شعار
  - قسم المؤشرات الرئيسية (KPIs)
  - قسم الرسوم البيانية
  - قسم المستشفيات
  - قسم تفاصيل المستشفى
  - تذييل الصفحة

### 2. dashboard.css
- **الوصف**: ملف التنسيقات المخصصة
- **المحتوى**: جميع الأنماط الخاصة بالداشبورد
- **الأقسام**:
  - أنماط تفاصيل المستشفى
  - شارات الحالة والأولوية
  - شارة البلاغات الحمراء
  - تنسيق عناصر البلاغات
  - متغيرات الألوان
  - تحسينات الاستجابة
  - تأثيرات إضافية

### 3. dashboard.js
- **الوصف**: ملف JavaScript الرئيسي
- **المحتوى**: جميع الوظائف والتفاعلات
- **الأقسام**:
  - بيانات المستشفيات والبلاغات
  - وظائف إدارة بطاقات المستشفيات
  - وظائف إدارة البلاغات الحمراء
  - وظائف تفاصيل المستشفى
  - وظائف الرسوم البيانية
  - معالجات الأحداث
  - وظائف التهيئة

## كيفية التعديل

### تعديل البيانات
```javascript
// في ملف dashboard.js
const hospitalsData = [
  {
    id: 1,
    name: "اسم المستشفى",
    type: "نوع المستشفى",
    // ... باقي البيانات
  }
];
```

### تعديل الألوان
```css
/* في ملف dashboard.css */
:root {
  --alert-red: #ef4444;      /* لون التنبيه الأحمر */
  --alert-orange: #f59e0b;   /* لون التنبيه البرتقالي */
  --primary-blue: #002B5B;   /* اللون الأزرق الأساسي */
}
```

### تعديل التخطيط
```html
<!-- في ملف dashboard.html -->
<section class="py-16 mesh-gradient">
  <!-- محتوى القسم -->
</section>
```

## الميزات

### 1. التنظيم
- فصل كامل بين HTML, CSS, و JavaScript
- تعليقات مفصلة باللغة العربية
- هيكل واضح ومنطقي

### 2. سهولة التعديل
- كل قسم له تعليق واضح
- متغيرات منفصلة للبيانات
- وظائف منظمة ومستقلة

### 3. الاستجابة
- تصميم متجاوب لجميع الأجهزة
- تحسينات خاصة للشاشات الصغيرة
- تأثيرات بصرية جذابة

### 4. التفاعل
- رسوم بيانية تفاعلية
- فلترة المستشفيات
- عرض تفاصيل المستشفيات
- تنبيهات البلاغات الحرجة

## المتطلبات

### المكتبات الخارجية
- **Tailwind CSS**: للتنسيقات الأساسية
- **Chart.js**: للرسوم البيانية
- **خط Tajawal**: للخط العربي

### الملفات المحلية
- `../public/assets/css/styles.css`: ملف التنسيقات الأساسي
- `../public/assets/js/app.js`: ملف JavaScript الأساسي
- `../public/assets/img/logo3.png`: شعار الموقع

## الاستخدام

1. افتح ملف `dashboard.html` في المتصفح
2. تأكد من أن جميع الملفات المطلوبة موجودة
3. استخدم الملفات المنفصلة للتعديل حسب الحاجة

## نصائح للتطوير

### إضافة مستشفى جديد
```javascript
// أضف في مصفوفة hospitalsData
{
  id: 7,
  name: "اسم المستشفى الجديد",
  type: "نوع المستشفى",
  beds: 200,
  totalReports: 100,
  openReports: 10,
  closedReports: 90,
  resolutionRate: 90,
  priorityCounts: { red: 0, orange: 2, yellow: 5 },
  redReports: []
}
```

### إضافة نوع بلاغ جديد
```javascript
// أضف في مصفوفة sampleReports
{
  id: 'RPT-2025-XXX',
  type: 'نوع البلاغ الجديد',
  hospital: 'اسم المستشفى',
  priority: 'red', // أو 'orange' أو 'yellow'
  date: '2025-01-16 10:00'
}
```

### تخصيص الألوان
```css
/* أضف ألوان جديدة في :root */
:root {
  --custom-color: #your-color;
}

/* استخدم اللون الجديد */
.custom-element {
  color: var(--custom-color);
}
```
